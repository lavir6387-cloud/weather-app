# 🌦️ Weather App

A simple Weather App built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Shows temperature
- Shows humidity
- Shows wind speed
- Shows weather icon
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
